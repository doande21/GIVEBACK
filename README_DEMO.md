
# GIVEBACK - Báo cáo Dự án Công nghệ

## 1. Giới thiệu dự án
GiveBack là nền tảng kết nối cộng đồng thiện nguyện, giúp tối ưu hóa quy trình tặng đồ cũ và quyên góp từ thiện thông qua công nghệ.

## 2. Các tính năng chính
- **AI Vision Scanning**: Tự động nhận diện và phân loại đồ vật từ hình ảnh.
- **Bản đồ thông minh**: Tìm kiếm điểm tiếp nhận từ thiện gần nhất.
- **Quản lý sứ mệnh**: Theo dõi tiến độ quyên góp thời gian thực.
- **Đấu giá từ thiện**: Gây quỹ thông qua các vật phẩm giá trị.
- **Cộng đồng**: Chia sẻ khoảnh khắc và lan tỏa yêu thương.

## 3. Công nghệ sử dụng
- **Frontend**: React 19, TypeScript, Tailwind CSS.
- **Backend**: Firebase (Firestore, Auth).
- **AI**: Gemini API (Flash 3.0) cho Vision và Chatbot.
- **Animation**: Motion (framer-motion).

## 4. Hướng dẫn Demo
- Truy cập vào ứng dụng.
- Nhấn vào biểu tượng **Ngôi sao (Showcase)** trên thanh điều hướng để xem trang giới thiệu dự án dành cho báo cáo.
- Thử nghiệm tính năng **Tặng đồ** để thấy AI Vision hoạt động.
- Kiểm tra **Bản đồ** để thấy Maps Grounding.
